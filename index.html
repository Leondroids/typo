Introduction

Initialize Typo with an Android context and provide a FontMapper

Typo.init(context).withFontMapper(MyFontMapper)

and use it in a layout file

<com.leondroid.typo.views.TypoTextView xmlns:custom="http://schemas.android.com/apk/res-auto" android:layout_width="wrap_content" android:layout_height="wrap_content" android:text="Extra Light Italic" custom:typoName="@string/my_font_name" custom:typoStyle="@integer/extra_light_italic" />
